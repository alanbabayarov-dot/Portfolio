# ⚡ Vector D — AI Tools & Tech Pipeline

> Цель — не быть дизайнером, который «использует AI», а строить конкурентные преимущества через скорость, масштаб и визуальный интеллект.

---

## CORE AI ПАЙПЛАЙН

```
Идея → Prompt System → Генерация → Отбор → Бриф → Применение

Целевое время: 30–45 минут от идеи до визуального брифа
```

---

## ИНСТРУМЕНТЫ

### Генерация изображений
| Инструмент | Применение |
|---|---|
| **Flux Kontext Dev** | Основная генерация для Euler и Toolor |
| **Flux Fill + BiRefNet** | Inpainting, удаление фона, изоляция продукта |
| **Midjourney** | Pre-shoot визуальные брифы, генерация мудбордов |
| **SDXL + XY Tiling** | Генерация принтов и паттернов |

### Примерка и видео
| Инструмент | Применение |
|---|---|
| **IDM-VTON / CatVTON / Qwen VTO** | Virtual try-on пайплайны |
| **Wan 2.2 VACE** | Генерация видео для кампаний |
| **Wan 2.2 Video Restyle** | Стилизация видео |
| **Wan 2.6 Flash** | Быстрый видеоконтент |
| **Florence-2 + SAM** | Сегментация изображений и маски |

### Среда
| Инструмент | Применение |
|---|---|
| **ComfyUI** | Основная среда для всех AI пайплайнов |
