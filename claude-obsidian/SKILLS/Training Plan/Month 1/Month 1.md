# 📅 Месяц 1 — Art Direction + Silhouette + Leadership

> **Цель:** Понять анатомию кампании как оператор. Иметь мудборд + Visual Tone Guide + Lookbook 1. Запустить персональный крокис. Установить систему брифинга команды.

---

## НЕДЕЛЯ 1 — Анатомия кампании + Архетипы силуэтов

**Ежедневно:**
- Нечётные дни → 1 editorial фрейм (Rick Owens, Craig Green, Stone Island)
- Чётные дни → 10 архетипов силуэтов (A, H, X, O, T, Y), по 2 каждого

**Среда:**
- [ ] Разобрать 2 кампании по структуре: концепт → кастинг → свет → локация → пост
- [ ] Написать запись в Design Journal #1
- [ ] Посмотреть: интервью Fabien Baron на YouTube
- [ ] Написать честную самооценку по всем 4 векторам — текущий уровень

**Воскресенье:**
- [ ] Построить первый мудборд Euler — инструмент решений, не коллаж красивостей
- [ ] Deliverable: 1 мудборд + 1 страница комментария — почему каждый выбор
- [ ] Написать 5–7 принципов лидерства для построения команды

---

## НЕДЕЛЯ 2 — Свет, кастинг + Персональный крокис

**Ежедневно:**
- Фреймовые дни → только источник и качество света
- Скетчевые дни → 10 фигур-крокисов, только контур, поиск личной пропорции

**Среда:**
- [ ] Изучить Willy Vanderperre и Nick Knight — как они обращаются с моделью как с образом
- [ ] Написать: носитель Euler в 5 словах. Носитель Toolor в 5 словах
- [ ] Найти 3–5 реальных референсов лица для каждого
- [ ] Прочитать первую главу: Creativity Inc или Ego Is The Enemy

**Воскресенье:**
- [ ] Запустить Midjourney или Flux
- [ ] Построить первый AI pre-shoot visual brief для Euler: 6–8 фреймов с описаниями света, кастинга, тона
- [ ] Это первый AI-инструмент как арт-директор

---

## НЕДЕЛЯ 3 — Visual Tone Guide + Архитектура плеча

**Ежедневно:**
- Скетчевые дни → 5 фигур с одеждой как чёрный силуэт + 5 фигур-крокисов без одежды

**Среда:**
- [ ] Написать Visual Tone Guide для Euler: 10 правил + 5 запретов. Конкретно, не абстрактно
- [ ] Leadership: написать стандарт брифинга — каждое задание = контекст + задача + критерии успеха

**Воскресенье:**
- [ ] Изучить архитектуру плеча: dropped, structured, raglan, saddle
- [ ] Сравнить Balenciaga vs Rick Owens vs Acronym — три разные идеологии
- [ ] Написать Fit Philosophy Euler: 3 слова + конкретные параметры

---

## НЕДЕЛЯ 4 — Первый Lookbook + Первые силуэтные коды

**Ежедневно:**
- Скетчевые дни → только силуэты Euler. Правило: если форма не читается без деталей — перерисовывай

**Среда:**
- [ ] Написать полный бриф для первого lookbook'а: концепт, кастинг, локация, свет, 5 ключевых фреймов
- [ ] Leadership: провести первый 1-on-1 с участником команды. Задокументировать письменно

**Воскресенье:**
- [ ] Снять первый lookbook по написанному брифу
- [ ] 30–40 фреймов → выбрать 10–12 финальных
- [ ] Это портфолио-кусок, не эксперимент

---
